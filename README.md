# Interior Design Telegram Bot

Бот принимает **фото комнаты**, задаёт вопросы о стиле и переделывает интерьер через **FLUX.1-Kontext** (сохраняет планировку).

## Запуск

```bash
cd interior-telegram-bot
./start.sh
```

В Telegram: `/design` → **фото** → вопросы → новый дизайн.

## Команды

| Команда | Действие |
|---------|----------|
| `/start` | Приветствие |
| `/design` | Новый дизайн (сначала фото) |
| `/cancel` | Отмена |

## Переменные (.env)

- `HF_MODEL` — `black-forest-labs/FLUX.1-Kontext-dev` (image-to-image по фото)

## Важно

- Токены **не коммитьте** в git (`.env` в `.gitignore`)
- Если токены попали в чат — перевыпустите их в BotFather и Hugging Face
- Бесплатный HF: очередь, лимиты; первая генерация может занять до 2 минут
- Сейчас **редизайн по фото** (FLUX Kontext), планировка сохраняется

## Деплой 24/7

Render / Railway: `python bot.py` с переменными окружения из `.env`.
